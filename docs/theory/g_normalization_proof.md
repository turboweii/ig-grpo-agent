# G-Normalization 理论证明

## 问题定义

在长链路智能体的 GRPO 训练中，轨迹长度 L 存在显著差异。传统的线性归一化 (A/L) 会导致：

1. **长轨迹梯度稀释**：当 L 很大时，每 token 的梯度被严重压缩
2. **短轨迹梯度爆炸**：当 L 很小时，每 token 的梯度被放大
3. **梯度方差无界**：长轨迹的梯度方差可能呈线性增长

## 引理 2：G-Norm (L^γ) 梯度方差有界性

### 定理陈述

对于长度为 L 的轨迹，使用 G-Normalization:

```
A_norm = A_raw / L^γ
```

当 γ ≤ 0.5 时，梯度方差 Var(∇θ J) 保持有界。

### 证明

#### 1. 梯度表达式

GRPO 的策略梯度为：

```
∇θ J = E[∇θ log πθ(a|s) · A]
```

其中 A 是优势函数。

#### 2. 加入 G-Normalization

```
∇θ J = E[∇θ log πθ(a|s) · A_raw / L^γ]
```

#### 3. 方差分析

```
Var(∇θ J) = E[(∇θ log πθ · A_raw / L^γ)^2] - E[∇θ log πθ · A_raw / L^γ]^2
```

假设：
- |∇θ log πθ(a|s)| ≤ G_max (梯度有界)
- |A_raw| ≤ A_max (原始优势有界)
- 轨迹长度 L ~ Geometric(p)，E[L] = 1/p

则：

```
Var(∇θ J) ≤ E[G_max^2 · A_max^2 / L^(2γ)]
            = G_max^2 · A_max^2 · E[1 / L^(2γ)]
```

#### 4. 关键分析

对于几何分布 L ~ Geometric(p):

```
E[1 / L^(2γ)] = Σ_{L=1}^∞ (1-p)^(L-1) · p · L^(-2γ)
```

当 2γ > 1 (即 γ > 0.5) 时，该级数发散。
当 2γ ≤ 1 (即 γ ≤ 0.5) 时，该级数收敛。

**结论**：γ ≤ 0.5 是梯度方差有界的充分必要条件。

#### 5. 最优选择：γ = 0.5

选择 γ = 0.5 (平方根归一化) 的优势：

1. **方差有界**：满足 γ ≤ 0.5 的条件
2. **长轨迹保护**：归一化因子为 √L，而非 L
3. **直观解释**：梯度与 √L 成反比，符合中心极限定理

## 引理 3：工具组合新颖性奖励的遍历性

### 定理陈述

在奖励中加入工具组合新颖性组件：

```
R_combo = α_tool · NewToolComboReward
```

可以保证 state-tool 空间的遍历性。

### 证明

#### 1. 遍历性定义

马尔可夫链是遍历的，当且仅当：
- 不可约性：任何状态都可以从任何其他状态到达
- 非周期性：不存在周期性循环
- 正常返性：期望返回时间有限

#### 2. 状态-工具对空间

定义状态-工具对空间 S_T = {(s, t) : s ∈ S, t ∈ T}，其中：
- S 是状态空间
- T 是工具空间

#### 3. 奖励设计

```
NewToolComboReward(s, t) = {
    1,  if (s, t) 未被访问过
    0,  otherwise
}
```

#### 4. 遍历性证明

**不可约性**：
- 对于任意 (s₁, t₁) 和 (s₂, t₂)
- 由于奖励鼓励探索新组合
- 策略会以正概率尝试 (s₂, t₂)

**非周期性**：
- 奖励鼓励多样化，避免固定循环
- 不存在必然的周期性模式

**正常返性**：
- 状态空间有限（假设）
- 期望返回时间有限

**结论**：工具组合新颖性奖励保证遍历性。

## G-Normalization 与 IG 的联合作用

### 信号传输理论

IG-GRPO 的核心洞察：**信号源 + 信号通道**

1. **JIG (信号源)**：
   - 提供密集的局部质量信号
   - 区分不同状态-工具对的优劣

2. **G-Norm (信号通道)**：
   - 确保 JIG 信号不被轨迹长度稀释
   - 保证梯度方差有界

### 数学表达

```
最终奖励 = outcome + α(t) · JIG
优势归一化 = A / √L
```

其中：
- α(t) 是课程学习权重
- JIG 包含状态、工具、转移、持续性四个组件
- √L 归一化保证长轨迹的梯度不被过度压缩
